# 🦅 Brothers Network Finance - Sistema Master de Gestão

Sistema web integrado de alta performance desenvolvido para a **Brothers Network Finance**, centralizando a captura de membros, análise automatizada de perfil e gerenciamento de esteira de atendimento (CRM) em um ambiente seguro.

---

## 🚀 Funcionalidades Atuais

### 🌐 Visão Pública (Portal do Cliente)
*   **Landing Page de Elite:** Interface premium com identidade visual institucional (Dark/Gold), gatilhos mentais de autoridade, escassez e blocos de prova social.
*   **Termômetro de Score Interativo:** Slider de engajamento psicológico que gera diagnósticos automáticos com base na pontuação de crédito informada pelo cliente.
*   **Portal de Conteúdo (Blog):** Feed de artigos diários nativo, focado em educação financeira, direito bancário e networking.

### 📊 Visão Privada (Painel do Administrador)
*   **Sistema de Autenticação:** Tela de bloqueio restrita para sócios e equipe operacional através de login e senha.
*   **Esteira de Atendimento Dinâmica (CRM):** Painel interativo para gerenciar as 5 tarefas críticas de atendimento (`T1` a `T5`) com salvamento definitivo.
*   **WhatsApp Dinâmico:** Links gerados automaticamente com mensagens personalizadas que mudam de texto de acordo com a fase atual do cliente na esteira.
*   **Calendário de Relacionamento:** Monitoramento automatizado de aniversariantes do mês com gatilhos de mensagens comerciais.
*   **Dashboard de Performance (Analytics):** Gráficos e métricas em tempo real (KPIs de conversão, contagem de perfis críticos e volume de interesses).
*   **Painel de Imprensa:** Publicador de artigos que alimenta o blog da página inicial direto pelo painel de controle.

---

## 🛠️ Tecnologias Utilizadas

*   **Linguagem Principal:** Python 3
*   **Framework Web:** Streamlit (Interface de Alta Performance)
*   **Banco de Dados:** SQLite (Persistência local de dados integrada)
*   **Processamento de Dados:** Pandas
*   **Hospedagem:** Streamlit Community Cloud
